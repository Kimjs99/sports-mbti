## 주요 기능

- 🧠 **16문항 성격 검사** — 팀워크, 순발력, 지구력, 정밀성, 도전성, 계절 선호도 측정
- 🏅 **20개 올림픽 종목 매칭** — 하계 13종목 + 동계 7종목
- 📊 **레이더 차트 시각화** — 나의 운동 성향을 5가지 축으로 표시
- 🎓 **학번·이름 입력** — 수업 중 개인별 결과 확인 가능
- 📱 **반응형 디자인** — 모바일·태블릿·PC 모두 지원
- ⚡ **빠른 로딩** — 빌드 없이 단일 HTML 파일로 즉시 실행

## 사용 방법

### 학생용

```
1. 배포 URL 접속
2. [검사 시작하기] 버튼 클릭
3. 학번 (4자리 이상) · 이름 (2자 이상) 입력
4. 16개 질문에 순서대로 답변
5. 결과 확인 — 추천 종목 · 성향 설명 · 레이더 차트
6. [재도전하기] 로 다시 검사 가능
```

## 점수 산출 방식

### 측정 트레이트 (6가지)

| 트레이트 | 설명 | 최대 점수 |
|----------|------|-----------|
| `teamwork` | 팀워크 성향 | 14점 |
| `explosive` | 순발력·폭발력 | 14점 |
| `endurance` | 지구력·인내심 | 14점 |
| `precision` | 정밀성·집중력 | 14점 |
| `risk` | 도전성·위험 감수 | 12점 |
| `winter` | 동계 스포츠 선호도 | 8점 |

### 계산 흐름

```
1. 16문항 답변 수집
         ↓
2. 트레이트별 원점수 합산
         ↓
3. 0~10 범위로 정규화
   norm[k] = (raw + maxP/2) / maxP × 10
         ↓
4. 모든 종목과 유사도 계산 (유클리드 거리)
   similarity = -Σ(sport.trait[k] - norm[k])²
         ↓
5. 유사도 최대 종목 = 최종 결과
```

### 동계 편향 방지 설계

Q5 (계절 선호) 는 **대칭 배점** 적용:
```
❄️ 겨울 선택 → winter: +4
☀️ 여름 선택 → winter: -4  ← 동일한 절댓값
```

Q15 (동계 스포츠)는 **winter 점수 미부여** — 플레이 스타일만 반영:
```
스키·스케이팅 → explosive: +2, risk: +2
컬링·하키     → teamwork: +3, precision: +1
```

## 개발 과정

### 1단계 — 기획 및 데이터 설계
- 올림픽 종목 20개 선정 (하계 13 + 동계 7)
- 종목별 6개 트레이트 수치 설계
- 성격-스포츠 매칭 알고리즘 설계 (유클리드 거리 기반)

### 2단계 — 질문지 설계
- 16문항 구성 (사회성, 에너지, 계획성, 팀워크, 계절 선호 등)
- 각 문항별 트레이트 가중치 부여
- 동계 편향 테스트 및 점수 균형 조정

### 3단계 — UI 개발
- React + 인라인 스타일로 4단계 화면 구현
  - 랜딩 → 인적사항 입력 → 16문항 퀴즈 → 결과
- SVG 기반 레이더 차트 구현
- 올림픽 링, 플로팅 이모지 등 시각 요소 추가
- 동계/하계별 컬러 테마 분리 (파랑 / 금색)

### 4단계 — 최적화 및 배포
- Babel CLI로 JSX → 순수 JS 사전 컴파일
- 단일 `index.html` 파일로 번들링
- GitHub Pages 정적 배포

### 주요 수정 이력

| 버전 | 수정 내용 |
|------|-----------|
| v1.0 | 초기 React 앱 구현 |
| v1.1 | Canvas API 다운로드 기능 추가 |
